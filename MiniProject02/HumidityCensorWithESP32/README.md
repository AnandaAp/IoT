# ini adalah mini project kedua
## penggunaan esp32 untuk mendeteksi sensor
### berikut adalah komponen yang digunakan:
- 3 buah led
- 4 buah resistor 1k Ohm
- 1 buah button
- 1 buah sensor DHT 11
- 1 buah ESP-WROOM32
- kabel jumper seperlunya
